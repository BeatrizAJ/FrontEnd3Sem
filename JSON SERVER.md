**\*JSON SERVER \***

facilita o desenvolvimento front end criando um servidor

cria uma API falsa

para programadores Front-End não terem que esperar o programador Back-End finalizar a API utilizando um arquivo db.json



**\*Como instalar\***

abrir o terminal, pode ser no VSCode, e rode o comando:

npm install -g json-server

**criar um arquivo db.json (este arquivo serve como um banco de dados gerenciado Exemplo:)**

**{**

&#x20;   **"Rota/EndPoint?":\[**

&#x20;     **{"id": 1, "dado-qualquer": "valor qualquer"}**

**]**



**}**



**Exemplo fictício:**

**{**

&#x20;   **"usuarios?":\[**

&#x20;     **{"id": 1, "nome": "Bia", "Email": "Bia@gmail.com","idade": 16},**

&#x20;     **{"id": 2, "nome": "x", "Email": "x@gmail.com","idade": 10},**

**]**



**}**
